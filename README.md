# Election Commission Project – Bootstrap 5 Internship Task
## Project Overview
This project was developed as part of a Bootstrap 5 internship task with a focus on
**Election Commission and Democratic Awareness**.  
The aim of the project is to design clean, modern, and responsive web pages using
Bootstrap 5 while presenting information related to elections, voter awareness,
and the importance of democracy.
The project consists of multiple responsive pages:
- Home Page
- About Page
- Services Page
- Contact Page
All pages are built using **Bootstrap 5 via CDN** and follow responsive design principles.
## Step-by-Step Process
1. Carefully reviewed the internship task requirements.
2. Explored official Bootstrap 5 documentation and example layouts.
3. Planned the website structure based on an Election Commission theme.
4. Identified and used essential Bootstrap components such as:
   - Navbar
   - Hero section
   - Cards
   - Grid system
   - Forms
   - Footer
5. Created the basic project structure:
   - `index.html`
   - `about.html`
   - `services.html`
   - `contact.html`
6. Designed the Home page with:
   - Election-themed hero section
   - National flag background image
   - Importance of elections section
7. Designed the About page to explain the role and responsibilities of the Election Commission.
8. Designed the Services page to display voter-related services using Bootstrap cards.
9. Designed the Contact page with a responsive Bootstrap form for user interaction.
10. Ensured proper use of background images with overlays for text readability.
11. Ensured responsiveness across mobile, tablet, and desktop devices.
12. Uploaded the project to GitHub.
13. Deployed the website using GitHub Pages.

## Challenges Faced & Solutions
### Understanding Bootstrap Grid System
- Initially faced difficulty in structuring layouts using rows and columns.
- Solved by referring to Bootstrap grid documentation and practice examples.
### Managing Background Images
- Faced issues with image paths and overlays.
- Resolved by organizing assets properly and using CSS overlays.
### GitHub Pages Deployment
- Faced issues while enabling GitHub Pages.
- Fixed by selecting the correct branch and root folder in repository settings.
